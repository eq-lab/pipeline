## Format

Each plan lives at `docs/exec-plans/active/<feature>.md` while in progress. On completion, move it to `docs/exec-plans/completed/<feature>.md`.

Plans follow the AGENTS.md workflow: issue → spec → plan → docs → review → implement → test → archive → commit → PR.
